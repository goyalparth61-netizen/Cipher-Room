# ⚡ About Cipher Room

*"Where Artificial Intelligence transforms cybersecurity into an unforgettable escape experience."*

</div>

Cipher Room is a next-generation **AI-powered cyber escape room** designed to blend **learning**, **problem-solving**, and **real-world cybersecurity concepts** into a single immersive experience.

Unlike traditional escape room games with fixed puzzles, Cipher Room uses **Artificial Intelligence** to generate a **brand-new mission every time a player logs in**. Every investigation introduces unique challenges, hidden clues, encrypted messages, and digital evidence, making each playthrough completely different.

Whether you're a **student**, **cybersecurity enthusiast**, or **puzzle solver**, Cipher Room offers a hands-on environment to explore digital investigations while having fun.

# 🤖 Meet ARES AI

### **Autonomous Response & Escape System**

</div>

ARES AI is the intelligent core of Cipher Room.

It acts as your mission controller, storyteller, and digital guide by generating unique cyber investigations, adapting mission difficulty, providing contextual hints, and analyzing player progress in real time.

Instead of replaying the same escape room repeatedly, ARES AI ensures that **every login creates a fresh adventure**.

# 📖 Story

</div>

A confidential research facility has suffered a massive cyber attack.

Sensitive files have been encrypted, digital evidence has been scattered, and an unknown threat actor has infiltrated the system.

As a newly recruited cyber investigator, your mission is to work alongside **ARES AI** to uncover hidden clues, recover critical data, identify the attacker, and escape before the facility enters permanent lockdown.

Every investigation follows a different storyline, introducing new puzzles, fresh evidence, and unexpected twists.

# 📥 Clone Repository

```bash
git clone https://github.com/your-repository/Cipher-Room.git

cd Cipher-Room
```

---

# 📦 Install Dependencies

### 🎨 Frontend

```bash
cd client
npm install
```

### ⚙️ Backend

```bash
cd ../server
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a **.env** file inside the **server** folder.

```env
GEMINI_API_KEY=YOUR_API_KEY

SECRET_KEY=YOUR_SECRET_KEY

DATABASE_URL=sqlite:///cipherroom.db
```

---

# ▶️ Run the Project

### Backend

```bash
cd server

uvicorn main:app --reload
```

### Frontend

```bash
cd client

npm run dev
```
